# install
1、Using makefile directly:
```
   cd p7zip
   make 7z (OR make 7za OR make 7zr OR make sfx)
```
2、OR Using cmake directly:
```
   cd p7zip/CPP/7zip/CMAKE/
   mkdir build
   cd build
   cmake ..
   make 
```
